# AbCd Dev Hub Studio

A responsive multipage portfolio + services website built with HTML, CSS, and JavaScript.

## Purpose
Showcase my frontend projects, describe my services, and provide an easy way to contact me.

## Pages
- Home (`index.html`)
- About (`about.html`)
- Projects (`projects.html`)
- Services (`services.html`)
- Contact (`contact.html`)

## Features
- Responsive layout (mobile-first)
- Shared header/footer across pages
- Mobile navigation menu (JavaScript)
- Theme toggle saved with localStorage (JavaScript)
- Projects filter/search (JavaScript)
- FAQ accordion (JavaScript)
- Contact form validation (JavaScript)
